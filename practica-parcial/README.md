# 🎓 Práctica para el Examen Parcial — Comportamiento del Consumidor

Bienvenido a la plataforma de entrenamiento para el examen parcial de la asignatura **Comportamiento del Consumidor** en la UTEC.

Esta aplicación ha sido diseñada para simular la complejidad y estructura del examen real, permitiendo a los estudiantes practicar con casos de estudio dinámicos y recibir retroalimentación detallada.

## 🚀 Características de la Aplicación

- **Estructura Fiel al Examen:** Cada simulacro consta de 6 preguntas estructuradas (Identificación en tabla, Bucle de hábito, Análisis de Checkout, Nudges, CDJ y Diseño de Experimentos).
- **Casos de Estudio Variados:**
  - **FreshRun:** Delivery de comida saludable.
  - **UrbanFit:** Suscripción de gimnasios boutique.
  - **PetAmigo:** E-commerce de productos para mascotas.
- **Interactividad Avanzada:** Incluye dinámicas de *Drag & Drop* y análisis de mockups visuales.
- **Feedback Inmediato:** Al finalizar, se muestra una rúbrica detallada para comparar con las respuestas de redacción libre.
- **Puntaje Parcial:** El sistema calcula automáticamente los aciertos proporcionales.
